## macOS

### Basics

- **Package manager**: `brew`
- **Default shell**: `zsh`

### Shortcuts & Hotkeys

- **System settings**
  - Disable Caps Lock action
  - Map: Control → Command, Command → Control
- **Desktop & navigation**
  - Show Launchpad: Control-Option-A
  - Show desktop: Control-Option-D
  - Stage manager: Control-Option-S
  - Mission Control: Control-Up
  - Change input source: Control-Space
  - Spotlight search: Option-Space
  - Quit app: Command-Q

### Aerospace

- **Layout**
  - Screen 1: Default-1, Any-1, Note, Message
  - Screen 2 (main): Default-2, Any-2, 0, 1, 2, 3, ...
- **Shortcuts (Alt-*)**
  - Change workspace layout: Alt-/ , Alt-,
  - Floating window: Alt-Shift-Space
  - Centralize floating window: Alt-Shift-.
  - Fullscreen floating: Alt-Shift-Enter
- **Focus window (Alt + key)**
  - Left: Alt-A
  - Right: Alt-D
  - Up: Alt-W
  - Down: Alt-S
- **Move window (Alt-Shift + key)**
  - Left: Alt-Shift-A
  - Right: Alt-Shift-D
  - Up: Alt-Shift-W
  - Down: Alt-Shift-S
- **Join window (Control-Alt-Shift + key)**
  - Left/Right/Up/Down: A / D / W / S
- **Resize**
  - Alt--, Alt-=
- **Workspace**
  - Alt-<number>: switch workspace
  - Alt-Shift-<number>: move window to workspace
  - Alt-Tab / Alt-Shift-Tab: previous workspace / move workspace to another screen
- **Service mode**
  - Alt-R: reload config
  - Alt-F: flatten workspace tree
  - Alt-Backspace: close all but current window
